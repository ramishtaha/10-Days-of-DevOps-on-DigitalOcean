About This Project

Project 1: Static Website on DigitalOcean Droplet

Project Objective

Learn the fundamentals of cloud infrastructure by manually creating a DigitalOcean Droplet, connecting via SSH, and deploying a static website using Nginx. This hands-on experience teaches essential skills like server provisioning, SSH connectivity, web server configuration, and basic Linux system administration.

Difficulty Level

Beginner - Perfect for those new to cloud computing and DevOps. No prior experience with cloud platforms required, just basic command line knowledge.

Time Investment

Estimated completion time: 2-3 hours. This includes droplet creation, configuration, website deployment, and testing.

Cost Analysis

Approximately $0.03 for this project (assuming 3 hours of usage). Fully covered by DigitalOcean's $200 free credit for new users.

Technical Specifications

Infrastructure

  • Provider: DigitalOcean Droplet
  • Instance Type: Basic Plan (1 vCPU, 1GB RAM, 25GB SSD)
  • Operating System: Ubuntu 24.04 LTS
  • Network: Public IPv4 address

Web Server

  • Server Software: Nginx 1.24+
  • Protocol: HTTP (Port 80)
  • Document Root: /var/www/html
  • Configuration: Custom server blocks

Security

  • Firewall: UFW (Uncomplicated Firewall)
  • SSH Access: Key-based authentication
  • Open Ports: 22 (SSH), 80 (HTTP), 443 (HTTPS)
  • SSL/TLS: Ready for future implementation

Frontend

  • HTML: HTML5 semantic markup
  • CSS: Modern CSS3 with Flexbox/Grid
  • Icons: Font Awesome 6.0
  • JavaScript: Vanilla JS for interactions

Learning Outcomes

Cloud Infrastructure Fundamentals

Understanding of Infrastructure as a Service (IaaS), virtual machines, and cloud networking concepts.

Linux System Administration

Hands-on experience with Ubuntu server management, package installation, and service configuration.

Command Line Proficiency

Improved skills in navigating Linux file systems, using SSH, and executing administrative commands.

Network Security

Understanding of firewall configuration, port management, and basic server security practices.

Web Server Configuration

Experience with Nginx installation, configuration, and basic performance optimization.

Deployment Process

Understanding the complete flow from infrastructure provisioning to application deployment.

Project Implementation Steps

1

Droplet Creation

Navigate DigitalOcean's interface to create and configure a new Ubuntu 24.04 droplet with appropriate specifications.

2

SSH Connection

Establish secure SSH connection to the droplet using key-based authentication or password.

3

System Updates

Update package repositories and upgrade system packages to latest stable versions.

4

Nginx Installation

Install and configure Nginx web server with automatic startup enabled.

5

Firewall Configuration

Configure UFW firewall to allow necessary ports while maintaining security.

6

Website Deployment

Create and deploy custom HTML, CSS, and JavaScript files to serve the static website.
